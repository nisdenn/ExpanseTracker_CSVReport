# 💰 Expense Tracker CLI

A simple command-line application to track income and expenses using Python.

## 🚀 Features
- Add income & expenses
- Store data in CSV file
- View transaction history
- Calculate total balance

## 📦 Tech Stack
- Python 3
- CSV (for storage)

## ▶️ How to Run

```bash
python main.py
````

## 📁 File Structure

```
expense-tracker/
│── main.py
│── expenses.csv (auto-generated)
```

## 💡 Example Usage

* Add salary → income
* Add food → expense
* Check balance anytime
